# 開発項目リスト（優先度付き）

## 【優先度1: 必須・コア機能】

### 1.1 基盤実装
- [ ] **プロジェクト構造作成** - ディレクトリ構造とファイル準備
- [ ] **依存関係設定** - requirements.txt, 環境構築
- [ ] **設定管理** - config/settings.py (API key等)
- [ ] **プロンプトローダー** - prompts/prompt_loader.py

### 1.2 PDF処理
- [ ] **PDFテキスト抽出** - services/pdf_processor.py
- [ ] **PDF画像抽出** - 図表処理対応（Vision機能用）
- [ ] **トークンカウント** - 250k制限チェック
- [ ] **ファイルサイズ検証** - 50MB制限

### 1.3 Semantic Kernel統合
- [ ] **kernel_service.py** - SK初期化、LLM接続、オーケストレーション
- [ ] **chat_manager.py** - チャット履歴管理
- [ ] **エージェント基底クラス** - agents/base_agent.py

### 1.4 エージェント実装
- [ ] **要約エージェント** - summarizer_agent.py
- [ ] **生徒エージェント** - student_agent.py  
- [ ] **先生エージェント** - teacher_agent.py
- [ ] **プロンプト統合** - .ini読み込み機能

### 1.5 基本UI
- [ ] **メインアプリ** - app.py Streamlit基本構造
- [ ] **ファイルアップロード** - PDFアップロード機能
- [ ] **要約表示** - 基本的な要約表示
- [ ] **Q&A表示** - ストリーミング表示

## 【優先度2: 重要機能】

### 2.1 Q&Aロジック
- [ ] **文書分割ロジック** - セクション均等分割アルゴリズム

- [ ] **フォローアップ判定** - 専門度閾値判定ロジック
- [ ] **ターン数制御** - 5-20回設定対応

### 2.2 UI改善
- [ ] **タブ切り替え** - ui/tabs.py（要約・Q&A / 最終レポート）
- [ ] **ストリーミング改善** - リアルタイム表示最適化
- [ ] **進捗表示** - 処理状況可視化
- [ ] **基本スタイリング** - ui/styles.py

### 2.3 最終出力
- [ ] **マークダウン生成** - 整形エージェントによる最終レポート
- [ ] **マークダウンレンダリング** - Streamlitでのコンパイル表示
- [ ] **コピーボタン** - レポートコピー機能

### Phase 3: 改善・拡張

- エラーハンドリング、UX改善、最適化
